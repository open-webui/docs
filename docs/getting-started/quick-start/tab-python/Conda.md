
# Install with Conda

1. **Create a Conda Environment:**

   ```bash
   conda create -n open-webui python=3.11
   ```

2. **Activate the Environment:**

   ```bash
   conda activate open-webui
   ```

3. **Install Open WebUI:**

   ```bash
   pip install open-webui
   ```

4. **Start the Server:**

   ```bash
   open-webui serve
   ```
