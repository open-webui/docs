# Notes, memos, and sharing

Workspaces aren't just for code. Notes, recordings, and things you send in from other apps all land as real files in the project folder: searchable, versionable, and visible to the AI like everything else.

## Markdown notes

Create a note and it's a Markdown file in the workspace. Edit it in the editor, preview it rendered, commit it with the project. There's no separate notes silo to export from later.

## Voice memos

Record a voice memo and the audio is saved in the workspace. When speech-to-text is configured, the recording is auto-transcribed to Markdown, so a thought captured while walking becomes a text file you can search and edit at your desk. Set up transcription in [voice and audio](/ecosystem/computer/ai/voice-and-audio).

## Share from other apps

Install Computer as a PWA on your phone and it appears in the OS share sheet. Share a link, text, an image, or a PDF from any app, and choose what it becomes:

- a **chat draft**: ready to send to the AI with your question
- a **note**: saved as a file in the workspace
- a **file import**: dropped into a workspace folder

Set the default action and the import destination folder in **Settings → PWA**. Installing the app and phone workflows in general are covered in [the phone app](/ecosystem/computer/phone-and-remote/phone-app).

## The always-works fallback

No share sheet, or on a device where the PWA isn't installed? Upload through the file browser: drag and drop or the upload action puts the file straight into the workspace from any signed-in browser.
