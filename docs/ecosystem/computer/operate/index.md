# Operate

Keep a long-running Computer instance healthy: know where its data lives, back it up, and check its logs when something looks off.

- [Data and backups](/ecosystem/computer/operate/data-and-backups): what is stored in the data directory and how to copy it safely.
- [Logs, health, and diagnostics](/ecosystem/computer/operate/logs-and-health): the health endpoint, log levels, and audit logging.

Upgrades are covered in [Updating](/ecosystem/computer/install/updating), and surviving reboots and sleep in [Keep it running](/ecosystem/computer/phone-and-remote/keep-it-running).
