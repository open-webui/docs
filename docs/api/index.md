# API Reference

Open WebUI provides a comprehensive REST API for programmatic access to all features.

## Base URL

```
http://localhost:3000/api
```

## Authentication

Include your API key in the Authorization header:

```bash
Authorization: Bearer YOUR_API_KEY
```

Generate API keys from **Settings > Account > API Keys**.

## Endpoints

See the [API Endpoints](./endpoints) page for detailed endpoint documentation.
